# File Uploader

A simple, opinionated web app for authenticated file storage and management with a lightweight UI.

Demo: https://file-uploader-m06o.onrender.com

Features
- User accounts and sessioned authentication.
- Multi-level folder hierarchy (create, edit, delete).
- File uploads, downloads and detail view with metadata (size, upload time).
- Breadcrumb navigation reflecting current path.
- Responsive, single light-orange theme with accessible controls and centered form layouts.
- Grid of folders and files with constrained cards and truncated filenames.
- Server-side file sanitization and safe storage keys.
- Storage backed by Supabase (object storage) and Prisma ORM for database access.
- Simple error reporting surfaced in the UI.
